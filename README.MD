🌐 **Site alvo dos testes:**  
https://carlosfelixpenha-create.github.io/QAPlayground/index.html

# 🏗️ EM CONSTRUÇÃO

## 🧪 Objetivo

Este repositório foi criado para:

- Praticar automação de testes com Cypress  
- Simular interações reais de usuário  
- Explorar diferentes cenários de teste  
- Servir como portfólio de estudos em QA  

---

## 🚀 Instalação

Clone o repositório:

```bash
git clone https://github.com/eweerfr/Testes-Cypress-QAPlayground.git
cd Testes-Cypress-QAPlayground
```

Instale as dependências:

```bash
npm install
```

---

## ▶️ Executando os Testes

### 🔎 Modo interativo (Interface gráfica)

```bash
npx cypress open
```

Abre o Test Runner do Cypress para executar os testes visualmente.

---

### ⚡ Modo headless (Terminal / CI)

```bash
npx cypress run
```

Executa todos os testes via linha de comando, ideal para integração contínua.
